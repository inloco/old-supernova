Typography

Set $customizable-typography: true and configure all settings below:

  • FONT SETTINGS
  • $customizable-font-family
  • $customizable-font-monospace
  • $customizable-font-size
  • $customizable-line-height
  • $customizable-headings-line-height
  • $customizable-font-weight
  • HEADING
  • $customizable-main-heading-size
  • $customizable-scale-factor
  • HORIZONTAL / VERTICAL MOVEMENTS
  • $customizable-horizontal-movement-divider
  • $customizable-vertical-movement-divider
  • --
  • Choose a font-family. Default is Roboto.
  • Choose a monospaced font. Default is Menlo.
  • Configure font-size. Use only px, em or % units - Default is 16px.
  • Configure document line height. Use only numbers.
  • Configure headings line height. Use only numbers.
  • Configure default font-weight. 400 (regular) is default.
  • --
  • Set the size scale of biggest header. Default is 4.
  • Scale factor for headings. Default is 1.41421 ($augmented-fourth).
  • --
  • Set horizontal movement divider (font-size / ?). Default is 4.
  • Set vertical movement divider (line-height / ?). Default is 4.
